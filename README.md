## CRUDInSlim ##

Le front-end affiche la liste des utilisateurs,  puis une page de détails pour chacun d’eux. Il est possible de faire toutes les actions de bases sur
les utilisateurs :
créer,
lire,
mettre
à
jour,
supprimer. 

BDD: mysql
